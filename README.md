# سیستم مدیریت مرکز آموزشی - نسخه پیشرفته 2.1.0

## 🎯 خلاصه پروژه

سیستم مدیریت مرکز آموزشی یک پلتفرم جامع و پیشرفته برای مدیریت کامل مراکز آموزشی است که شامل مدیریت کاربران، برنامه‌ها، حضور و غیاب، نمرات، نظرسنجی‌ها و گزارش‌گیری می‌باشد. این سیستم با قابلیت اتصال به Supabase و پشتیبانی از LocalStorage، انعطاف‌پذیری بالایی را فراهم می‌کند.

## 🚀 نصب و راه‌اندازی

### پیش‌نیازها
- مرورگر مدرن (Chrome، Firefox، Safari، Edge)
- اتصال اینترنت (برای استفاده از Supabase)
- یا بدون اینترنت (برای استفاده از LocalStorage)

### نصب سریع
1. **دانلود پروژه**:
   ```bash
   git clone https://github.com/your-repo/educational-center-system.git
   cd educational-center-system
   ```

2. **راه‌اندازی Supabase (اختیاری)**:
   - ایجاد حساب در [Supabase](https://supabase.com)
   - ایجاد پروژه جدید
   - اجرای فایل `supabase-schema.sql` در SQL Editor
   - کپی کردن URL و Key در `supabase-config.js`

3. **اجرای پروژه**:
   ```bash
   # برای استفاده از LocalStorage
   python -m http.server 8000
   
   # یا استفاده از Live Server در VS Code
   ```

4. **دسترسی به سیستم**:
   - باز کردن `http://localhost:8000` در مرورگر
   - یا باز کردن `index.html` مستقیماً

### راه‌اندازی Supabase (پیشنهادی)

#### مرحله 1: ایجاد پروژه Supabase
1. به [Supabase Dashboard](https://app.supabase.com) بروید
2. روی "New Project" کلیک کنید
3. نام پروژه و رمز عبور دیتابیس را وارد کنید
4. منطقه مناسب را انتخاب کنید
5. روی "Create new project" کلیک کنید

#### مرحله 2: تنظیم دیتابیس
1. در پروژه جدید، به بخش "SQL Editor" بروید
2. محتوای فایل `supabase-schema.sql` را کپی کنید
3. روی "Run" کلیک کنید تا جداول ایجاد شوند

#### مرحله 3: تنظیم فایل پیکربندی
1. فایل `supabase-config.js` را باز کنید
2. `SUPABASE_URL` و `SUPABASE_ANON_KEY` را از بخش "Settings > API" کپی کنید
3. مقادیر را در فایل جایگزین کنید

#### مرحله 4: تنظیم Storage
1. به بخش "Storage" در Supabase بروید
2. یک bucket جدید با نام `documents` ایجاد کنید
3. تنظیمات دسترسی را برای آپلود فایل تنظیم کنید

## 🔧 عیب‌یابی

### مشکلات رایج

#### مشکل: اتصال به Supabase برقرار نمی‌شود
**راه‌حل**:
1. بررسی اتصال اینترنت
2. بررسی صحت URL و Key در `supabase-config.js`
3. بررسی تنظیمات CORS در Supabase

#### مشکل: فایل‌ها آپلود نمی‌شوند
**راه‌حل**:
1. بررسی تنظیمات Storage در Supabase
2. بررسی اندازه فایل (حداکثر 50MB)
3. بررسی فرمت فایل

#### مشکل: LocalStorage پر شده
**راه‌حل**:
1. پاک کردن کش مرورگر
2. حذف داده‌های غیرضروری
3. استفاده از Supabase برای ذخیره‌سازی
