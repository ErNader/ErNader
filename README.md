# سیستم مدیریت مرکز آموزشی - نسخه پیشرفته

یک وب‌سایت کامل و پیشرفته برای مدیریت مرکز آموزشی با قابلیت‌های مختلف برای کاربران مختلف و اتصال کامل به دیتابیس.

## ✨ ویژگی‌های جدید و بهبود یافته

### 🔗 اتصال کامل به دیتابیس
- **Supabase Integration**: اتصال کامل به دیتابیس Supabase
- **LocalStorage Fallback**: کارکرد کامل در حالت آفلاین
- **Real-time Data**: به‌روزرسانی لحظه‌ای داده‌ها
- **Advanced Queries**: پرس‌وجوهای پیشرفته و بهینه

## 🚀 نحوه استفاده

### حالت لوکال (بدون Supabase)
1. فایل‌ها را در یک پوشه قرار دهید
2. فایل `index.html` را در مرورگر باز کنید
3. سیستم با LocalStorage کار می‌کند
4. داده‌های نمونه به طور خودکار ایجاد می‌شوند

### حالت کامل (با Supabase)
1. فایل‌ها را در یک پوشه قرار دهید
2. مراحل نصب Supabase را از فایل `SUPABASE_SETUP.md` دنبال کنید
3. فایل `supabase-schema.sql` را در Supabase اجرا کنید
4. فایل `index.html` را در مرورگر باز کنید
5. سیستم با دیتابیس Supabase کار می‌کند

## 📁 ساختار فایل‌ها

```
├── index.html              # صفحه اصلی
├── register.html           # صفحه ثبت‌نام
├── login.html              # صفحه ورود
├── programs.html           # صفحه برنامه‌ها
├── dashboard.html          # صفحه داشبورد
├── style.css               # استایل‌های پیشرفته
├── script.js               # کدهای JavaScript پیشرفته
├── supabase-config.js      # پیکربندی و توابع Supabase
├── supabase-schema.sql     # ساختار کامل دیتابیس
├── SUPABASE_SETUP.md       # راهنمای نصب Supabase
└── README.md               # راهنمای کامل استفاده
```
